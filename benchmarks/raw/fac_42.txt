<h2 class="header"><a name="research">Research interests</a></h2>
<p>
	  I am working on static analysis for the verification of semantic
          properties of programs.
	  My main focus is on abstract interpretation and more specifically
          on symbolic abstractions:
	</p>
<ul>
<li> Static Analysis of embedded, safety critical software<br/>
<a href="http://www.astree.ens.fr/">the Astrée project</a>:
            static analysis,
            for proving the absence of runtime errors in safety critical
            embedded software. <br/>
            Commercial support by <a href="http://www.absint.com/">AbsInt</a>
            (see <a href="http://www.absint.com/astree/">here</a> for more
            info).
	  </li>
<li> Shape analysis <br/>
<ul>
<li> <a href="memcad.html">MemCAD ERC Project</a> </li>
<li> <a href="http://xisa.cs.colorado.edu/">the Xisa
                  analyzer</a> </li>
</ul>
</li>
<li> Certified Compilation <br/>
<a href="lcertify.html">the Lcertify translation validator</a>
</li>
<li> Abstract Interpretation
	  </li>
<li> Slicing, dependence semantics
	  </li>
<li> Verification of Spreadsheet Applications </li>
<li> Static Analysis of JavaScript Software </li>
</ul>
