</p><p>
<b>Interests:</b> My research is in programming language theory, and I'm
 also interested in logic and category theory. I've recently been especially interested in
 two kinds of programming language:
</p><ul>
<li><i><b>Probabilistic programming languages</b></i>:
  This is broadly to do with specifying and understanding statistical models using programming languages.
  I was recently awarded an <a href="https://cordis.europa.eu/project/id/864202">ERC Consolidator grant</a> in this area, and we recently 
  won a <a href="https://research.fb.com/announcing-the-winners-of-the-probability-and-programming-research-awards/">Facebook research award</a> too.
  My paper in
  <a href="#esop17">ESOP 2017</a> introduced the idea of using <i>s-finite kernels</i> as a simple compositional formalism allowing both sampling and observations.
  In <a href="#lics17a">LICS 2017</a> we introduced <em>quasi-Borel spaces</em>: a new setting for higher order probability theory; this is then used to validate some inference algorithms in <a href="#popl18">POPL 2018</a> and extended to recursion in <a href="#popl19">POPL 2019</a>. Our <a href="#pps17">PPS
    2017 abstract</a> is about the relationship between exchangeability and program modules;
  we followed this up with an analysis of the Beta-Bernoulli process in <a href="#icalp18">ICALP 2018</a>.
<!--  We had an earlier paper about semantics in <a href="#lics16">LICS 2016</a>. -->
<p></p></li>
<li><i><b>Quantum computing and programming languages</b></i>:
Most recently, Mathieu Huot and I found a neat universal property for quantum theory (<a href="#lics19">LICS 2019</a>, <a href="#qpl18">QPL 2018</a>) based on my theory of quantum programs
(<a href="#popl15">POPL 2015</a>). 
I've also been working on contextuality and non-locality with 
<a href="http://www.cs.ru.nl/~suijlen/">Sander Uijlen</a>
(see our <a href="#iandc18">ICALP 2015</a> paper), and on
quantum programming languages with 
<a href="http://www.cs.ru.nl/~mathysr/">Mathys Rennela</a>
(see our <a href="#mfps17">MFPS 2017</a> and <a href="#mfps15">2015</a> papers
and our <a href="#qpl16">QPL 2016</a> paper). My research in this area has been supported by my grant 'Quantum
Computation as a Programming Language' (<a href="http://gow.epsrc.ac.uk/NGBOViewGrant.aspx?GrantRef=EP/N007387/1">EP/N007387/1</a>).
</li>
</ul>
Aside from these recent interests, I've been involved in various foundational
aspects of programming language theory.
One key theme has been <b>computational monads and algebraic theories:</b>
This line of work stems from the observation that
notions of computation can be determined by program equations
using techniques from universal algebra. 
I have presented
new algebraic frameworks to analyze program equations
for quantum computation (<a href="#popl15">POPL 2015</a>), jumps (<a href="#lics14a">LICS 2014</a>),
local store and pi-calculus (<a href="#lics13">LICS 2013</a>, <a href="#fossacs10">FOSSACS 2010</a> and <a href="#mfps09">MFPS
  2009</a>)
and functional logic programming (<a href="#fossacs13">FOSSACS
  2013</a>).
See <a href="#fossacs19">FOSSACS 2019</a> for a program logic,
and <a href="#freyd-lawvere14">WACT 2014</a> for a categorical perspective.
I've also been involved in game semantics
(<a href="#lics14b">LICS 2014</a>) and
categorical operational semantics (e.g. <a href="#lics08">LICS
  2008</a>, <a href="#lics06">LICS 2006</a>, <a href="#lmcs11">LMCS 2011</a>).
