	    productivity.

<div id="Tools">
<h3>Tools</h3>
<p class="bodytext">
	      As a byproduct of my research, I have developed and
	      contributed to a number of tools.
	    </p>
<ul class="list-group">
<li class="list-group-item">
<div class="content">
<h4><a href="software/grasshopper">GRASShopper</a></h4>
                  A verification tool that checks functional correctness of
                  programs manipulating heap-allocated data
                  structures.
                </div>
</li>
<li class="list-group-item">
<div class="content">
<h4><a href="software/vermeer">Vermeer</a></h4>
                  An automated debugging tool that traces and explains
                  faults in C programs.
                </div>
</li>
<li class="list-group-item">
<div class="content">
<h4><a href="http://pub.ist.ac.at/~zufferey/picasso">Picasso</a></h4>
	          A static analyzer for depth-bounded concurrent
	          systems. It has been used to verify properties of
	          non-blocking concurrent data structures and distributed
	          message passing algorithms with an unbounded number of
	          threads and messages.
                </div>
</li>
<li class="list-group-item">
<div class="content">
<h4><a href="http://lara.epfl.ch/dokuwiki/doku.php?id=jahob_system">Jahob</a></h4>
                  A verification
	          system for programs written in a subset of Java. Using
	          Jahob, developers can statically prove that methods satisfy
	          their contracts in all possible executions, as well as that
	          they preserve essential structural invariants and design
	          constraints.
                </div>
</li>
<li class="list-group-item">
<div class="content">
<h4><a href="http://swt.informatik.uni-freiburg.de/tool/bohne">Bohne</a></h4>
                  A symbolic shape analysis
	          tool. It infers loop invariants of programs manipulating
	          heap-allocated data structures. Bohne is integrated into
	          the <a href="http://lara.epfl.ch/dokuwiki/doku.php?id=jahob_system">Jahob</a>
                  verification system.
                </div>
</li>
<li class="list-group-item">
<div class="content">
<h4><a href="http://research.microsoft.com/SLAyer">SLAyer</a></h4>
	          A tool for automatic formal verification of industrial low-level
	          software components. SLAyer is being developed at
                  <a href="research.microsoft.com/en-us/labs/cambridge/default.aspx">Microsoft
	            Research Cambridge</a>.
                </div>
</li>
<li class="list-group-item">
<div class="content">
<h4><a href="http://research.microsoft.com/projects/HAVOC/">HAVOC</a></h4>
                  A tool for specifying and
	          checking properties of systems software written in C, in the
	          presence of pointer manipulations, unsafe casts and dynamic
	          memory allocation. HAVOC is being developed
	          at <a href="research.microsoft.com/en-us/labs/redmond/default.aspx">Microsoft
	            Research Redmond</a>.
                </div>
</li>
</ul>
</div>
